API for a well known library known as jQuery known for its size and feature rich functionality. jQuery is known for it's simplification of the javascript language, allowing for less code to create interactivity and functionality. jQuery is free and as well as being an open source language(MIT) allowing for plugins that can be created on top of the library. reference link Javascript

Simplified definition of javascript detailing it's origins and its functionality. Javascript was developed by Netscape in order to add more dynamic content, as well as interactive elements. Though Javascript has the word "Java" in its namesake, it is not Java, though it is influenced by it in its syntax as well as being partially based on ECMAScript.

What is javascript
  1. Javascript is an essential ,interpreted, client sided scripting language created in 1995, used for dynamic interaction via on-the-fly calculations performed client side.
  2. Javascript is untyped, meaning "any operation can be performed on any data"
  3. viewed in the browser through developer tools, such as Firefox firebug or google Chrome Developer tools.
What it is not
  1. JS is not Java, although it takes much of its syntax from that language and Java's ancestor C.
  2. JS is not a server sided language, as all the processing takes place on the client side.
  3. JS is not a true class based Object Oriented programming language, although it does use prototyping.
