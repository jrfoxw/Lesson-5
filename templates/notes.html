What is javascript

  1. Javascript is an essential ,interpreted, client sided scripting language created in 1995, used for dynamic interaction via on-the-fly calculations performed client side.
  2. Javascript is untyped, meaning "any operation can be performed on any data"
  3. viewed in the browser through developer tools, such as Firefox firebug or google Chrome Developer tools.
What it is not
  1. JS is not Java, although it takes much of its syntax from that language and Java's ancestor C.
  2. JS is not a server sided language, as all the processing takes place on the client side.
  3. JS is not a class based Object Oriented programming language, although it does use prototyping

What is Android OS?

  1. Android OS is unique in that it is primarily designed for mobile devices, though it is also used on less mobile devices such as televisions.
  2. Android was initially designed in 2005 by the now defunct Android Inc. Later the product was bought by Google and released in 2007.
  3. Android is also unique (unlike the well known fruit phone) in that it's operating system, at its core, is open source, although proprietary software is normally developed on top of it.
  4. "Android is popular with technology companies that require a ready-made, low-cost and customizable operating system for high-tech devices."
